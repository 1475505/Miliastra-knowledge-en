# Miliastra-knowledge-en

English knowledge base for **Miliastra Wonderland** (千星奇域) Craftspeople Academy.

---

## Repository Structure

```
Miliastra-knowledge-en/
├── config/
│   ├── urls-guide.json          # Guide entry list (generated by spider/crawl)
│   ├── urls-tutorial.json       # Tutorial entry list (generated by spider/crawl)
│   └── title.json               # Title mapping table
├── official/
│   ├── guide/                   # Official guide Markdown (~186 pages, crawled)
│   ├── tutorial/                # Official tutorial Markdown (~76 pages, crawled)
│   └── translation/             # Community-translated pages (CN-only, see below)
├── derived/
│   ├── svg/                     # Generated SVG one-image-flow infographics
│   ├── svg_index.md             # Content list for SVG generation (69 entries)
│   └── svg_skill.md             # AI skill definition for SVG generation
└── spider/                      # Crawl & scrape toolchain (Node.js/TypeScript)
    └── README.md                # Crawl usage and SOP
```

## Content Sources

### `official/guide/` and `official/tutorial/`

Crawled from the **HoyoVerse English official documentation site** (`act.hoyoverse.com`).  
These files are generated automatically by the spider and should not be edited by hand.  
See [spider/README.md](spider/README.md) for crawl instructions and quality-check SOP.

### `official/translation/` — Community Translations

A small number of guide pages exist only in Chinese and have **not been officially published in English** by HoyoVerse. These pages return the Update Log when accessed on the English site.

Community-translated English versions of these pages are stored in `official/translation/`. They use the same frontmatter format as regular crawled files and can be used as source material for SVG generation.

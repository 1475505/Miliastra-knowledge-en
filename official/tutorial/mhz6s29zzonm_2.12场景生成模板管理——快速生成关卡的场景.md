### Feature Example 2: Configure Scene Generation Template Parameters to Create a Custom Scene

#### 1. Course Content

**Feature:** By adjusting various template parameters (e.g., Rock, Vegetation, Building), you can generate custom scenes tailored to specific needs or stylistic variations.

**Effect Demonstration:**

![Image 11](https://act-webstatic.hoyoverse.com/ugc-tutorial/course/sea/en-us/mheewlrpjhlq/8628f645-01e1-4590-8980-b27ceaedaf23.gif)

#### 2. Steps to Follow

**1. [In-Game Editor] Enter Scene Generation Template Management**

Click the the top-left corner of the in-game editor![Image 12](https://act-webstatic.hoyoverse.com/ugc-tutorial/course/sea/en-us/mheewlrpjhlq/bbcf8a72-10e2-4a81-ad71-35b1a66005f1.png) (or press Esc) >>> Bring out the menu on the left > Esc menu, scroll down and click [Scene Generation Template Management] >>> Open it.

![Image 13](https://act-webstatic.hoyoverse.com/ugc-tutorial/course/sea/en-us/mheewlrpjhlq/6819cd41-dca1-45a9-94ef-6fd16fb6e4f8.png)

**2. [In-Game Editor] Create a New Scene Generation Template**

In the management panel, create a new scene generation template.

![Image 14](https://act-webstatic.hoyoverse.com/ugc-tutorial/course/sea/en-us/mheewlrpjhlq/7d258def-4e48-4b46-a128-ce28196c385a.png)

**3. Using [In-Game Editor] to Apply Distribution Rules with the Selected Preset**

Click "Random Preview" below to preview the randomized result. If the distribution meets your expectations, click "Save Distribution Rules." Saved rules can be reused later by selecting them from the "Preset Distribution Rules List" when generating scenes.

![Image 15](https://act-webstatic.hoyoverse.com/ugc-tutorial/course/sea/en-us/mheewlrpjhlq/ac5c441f-a829-4f4e-b8f0-cb8ed368c725.png)

**3. [In-Game Editor] Modify Base Properties to Adjust the Generation Range**

Configure the zoom and rotation values to the desired settings.

![Image 16](https://act-webstatic.hoyoverse.com/ugc-tutorial/course/sea/en-us/mheewlrpjhlq/a82214e3-c262-445e-8a4d-cf6b512e04d3.png)

**4. [In-Game Editor] Adjust Asset Configuration to Update Scene Generation Output**

Modify the parameters within the selected layer. You can adjust generation density, randomization, and whether the layer is initially effective. The parameter behaviors are described below:

Parameter Name Parameter Type Parameter Effect
Initially Effective Boolean (Toggle)Enable This Layer When Applied
Generate Based on Terrain (Rock Layer)Boolean (Toggle)Defines whether generation is based on the bottom of the preview whitebox or on the terrain surface.
Generation Density Floating Point Controls how densely objects are generated within the same layer. Lower values produce sparser results, while higher values produce denser results.
Randomization Floating Point Controls the amount of jitter applied to sampling points during generation. A value of 0 results in a near-uniform distribution, while 1 produces a highly randomized distribution.
Association Range (Vegetation Layer)Floating Point Available only for the vegetation layer. Specifies the maximum distance from the vegetation target object within which generation occurs. Objects outside this range will not be generated.

Adjust the parameters and select the model assets you want to use.

![Image 17](https://act-webstatic.hoyoverse.com/ugc-tutorial/course/sea/en-us/mheewlrpjhlq/6b62927d-0158-4311-969a-43a29bae3f49.gif)

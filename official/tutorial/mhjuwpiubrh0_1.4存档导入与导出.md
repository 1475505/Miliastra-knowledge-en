# Save File Operations

## Save Page Settings

### How to Open

There are two ways of entering the save page based on your current state: [editor and Miliastra Sandbox closed — launch Genshin Impact and enter], or [editor and Miliastra Sandbox open — enter from editor and Miliastra Sandbox]

● **[Launch Genshin Impact and enter]:**After entering the game, click the Paimon Menu in the upper left corner![Image 2](https://act-webstatic.hoyoverse.com/ugc-tutorial/course/sea/en-us/mh3lyb9vkssa/43b4e072bfe84069818756872f9eddcf_1757675618.png) then click [My Miliastra Wonderland] to open the save page

![Image 3](https://act-webstatic.hoyoverse.com/ugc-tutorial/course/sea/en-us/mh3lyb9vkssa/c517dc316d3a45cf82ed66ddaf6a5257_1757675618.png)

**● [Enter from in-game editor]:**Click the upper left corner of the in-game editor![Image 4](https://act-webstatic.hoyoverse.com/ugc-tutorial/course/sea/en-us/mh3lyb9vkssa/9b356dec82874309a1988c691db123ff_1757675618.png) (or press Esc) >>> Bring out the menu on the left > Esc menu, click![Image 5](https://act-webstatic.hoyoverse.com/ugc-tutorial/course/sea/en-us/mh3lyb9vkssa/cbf7c622b0ff44dc8d64dcb9c2218d5d_1757675618.png) to open

![Image 6](https://act-webstatic.hoyoverse.com/ugc-tutorial/course/sea/en-us/mh3lyb9vkssa/940a295e9d384476b2576e1636792df5_1757675618.png)

### Feature Summary

**Save Page:**Supports save-related management such as save tab settings, specific save settings, and save file importing/exporting

![Image 7](https://act-webstatic.hoyoverse.com/ugc-tutorial/course/sea/en-us/mh3lyb9vkssa/6921afef168a4fd79fa5ff144cdd3345_1757675619.png)

[New Tab] and [Rename Save File] are the two most common operations. Importing/exporting will be explained further in detail below

**● New Tab:**Create a new save folder to categorize saves

**● Rename:**Edit save name to categorize and help memorize saves (only works on yourself, imported renamed saves will be named in the default format)

![Image 8](https://act-webstatic.hoyoverse.com/ugc-tutorial/course/sea/en-us/mh3lyb9vkssa/ee6a4744dc0249d09141ad5b9d622106_1757675619.gif)

## Save File Export

### Instructions

On the save page, select the save that needs to be saved >>> hover the mouse over it and press the lower left corner ![Image 9](https://act-webstatic.hoyoverse.com/ugc-tutorial/course/sea/en-us/mh3lyb9vkssa/4c843bc0ef0b4e0f9237a374f343de3b_1757675619.png)>>> click [Export Save File] on the opened menu >>> select a save path to export the save

_**The save naming on export and the aforementioned save renaming do not affect each other and can be set separately._

![Image 10](https://act-webstatic.hoyoverse.com/ugc-tutorial/course/sea/en-us/mh3lyb9vkssa/09bdcdb3f4644015a50b826049bbe3ca_1757675620.png)

The final exported save file is in [.gil] format:

![Image 11](https://act-webstatic.hoyoverse.com/ugc-tutorial/course/sea/en-us/mh3lyb9vkssa/b2eeb48d90a54a86a2ad4b082ac24113_1757675617.png)

On the save page, click [Import Save] in the bottom right corner >>> Select the save file you want to import

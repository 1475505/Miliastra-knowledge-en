# Combat Settings

# I. What are Combat Settings

Provides more detailed configuration for custom combat gameplay, enabling players to configure attack and hit response settings

# II. Editing Combat Settings

In the Specialized Settings tab, click _Advanced Editing_ to enter the entity's combat settings

![Image 2](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhhzhir8c5xi/774fc217-0ab8-406b-ad12-6b7118beca95.png)

Configuration Parameters Description
_Cannot Be Affected by Elemental Effects_ Configure whether this object can be affected by elemental effects
_Cannot Be Targeted_ Configures whether any function that targets and locks this object will take effect
_Cannot be targeted by allied units_ When enabled, this entity cannot be selected as a target by the following features:



Creation Autonomous Logic



Hitbox Attack



Ray Detection

Note: Even when this option is enabled, the entity can still be selected as a target for skill attacks
_Cannot be targeted by enemy units_ Same as above
_Bullet Tracking Point_ The name of the attachment point that bullets will track
_Hit Effect_ Configures special effect to play when the object is attacked
_Knockdown Effect_ Configures special effect to play when the object is destroyed or downed

The hit effects and knockdown effects can be further configured on the Edit Details tab

![Image 3](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhhzhir8c5xi/ddf7f3de-168f-4686-b27e-4103d5d1974a.png)

Effect Configuration Parameters Description
_VFX Asset Type_ _Only_ _Timed Effects_ _can be selected_

For effect-related assets, see [Special Effects](https://act.hoyoverse.com/ys/ugc/tutorial//detail/mhe1030vx380)
_Special Effects Assets_ You can select Special Effects Assets. After selecting a Special Effects asset, you can preview it in the scene![Image 4](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhhzhir8c5xi/4ff45317-664d-49d9-877a-397b877ee726.png)
_Zoom Factor_ Zoom factor of the effect
_Offset_ Specified Relative Position
_Rotation_ Specifies the rotation information of the special effect

# Main Interface

![Image 2](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhgli9r4nrni/6f9ec07c-44d0-4581-a62a-85f5e12c5979.png)

# I. System Menu

![Image 3](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhgli9r4nrni/1d817209-2533-4218-997f-4710390ef898.png)

Click the top-left menu icon (as shown) to open or close the System Menu, or press Esc to toggle it.

## 1. System Menu Functions

The System Menu provides the following functions:



Save current progress; open the save interface; upload a stage; open Miliastra Sandbox; and exit the Saved File.



Stage Settings, Manage UI Control Group, Manage Main Camera, Test Play, Multiplayer Test Play, Manage Peripheral System, Inventory, Currency, and Manage Wonderland Gift Box.



Manage Character Preview, Timer Management, Manage Stage Variable, Preset Point Management, Skill Resource Management, Manage Background Music, Manage Asset Import/Export, Manage Equipment Data, Shop Template Management, Advanced Data Management, Manage In-game Save, Manage Test Play Temporary Data, Manage Entity Deployment Groups, Manage Unit Tags, Shield Management, Scan Tag Management, Path Management, Manage Multi-language Text, Manage Light Source, Manage Text Chat, and Hotkey Overview.



View Creator (Craftsperson) Level and Creator (Craftsperson) Permissions.

## 2. Minimize

![Image 4](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhgli9r4nrni/729f2556-7efd-4bea-b55d-9701b95f69f3.png)

Interfaces opened from the System Menu collapse to the top bar. Left-click the corresponding function to reopen it.

# II. Assets Bar

![Image 5](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhgli9r4nrni/14d59f62-52fd-49ba-bbdf-2f3adecf0fcf.png)

Click the Assets Bar icon (top-left, as shown) to toggle on/off. Its contents vary by Edit Mode and typically include Custom and Preset items. Preset items cannot be edited directly; save them as Custom items to modify. The Assets Bar provides the following common functions:



Search: Search for content within the selected Assets Bar tab.



Uncategorized Tab: The default tab. It hides automatically when other Custom tabs exist and this tab contains no content.



Custom Tab Management: Create tabs, rename tabs, pin tabs, or dissolve tabs.

# III. Quick Settings

![Image 6](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhgli9r4nrni/58161e9c-f9b3-45cc-af20-707ed75bd5f3.png)

Click the Settings icon (top-right) to open Quick Settings, which include:



Camera Horizontal Speed: The horizontal movement speed of the camera in the editing interface.



Camera Vertical Speed: The vertical movement speed of the camera in the editing interface.



Horizontal Lock: When enabled, the camera moves parallel to the ground surface



Ground-Level Placement: When enabled, placed units are positioned flush with the terrain.



Surface Adsorption: When enabled, placed units will snap to nearby units.



Normalize Position After Surface Adsorption: Available only when Surface Adsorption is enabled. When enabled, units automatically rotate to align when snapping to other units.



Center Align: When enabled, the center point of the placed unit aligns automatically with nearby units.



Alignment Position: Specifies the coordinate system used for Center Align.



Show Distance When Moving: When enabled, displays the unit's relative movement at the bottom of the interface while moving.



Creation Entering Battle Range: Toggle whether to display the battle range for creations.



Always Show Preset Points: Toggle whether Preset Points are always shown.



Empty Object Hint Model Permanent Visibility: Toggle whether the indicator model for empty objects is always shown.



Always Show Path: Toggle whether paths are always shown.



Alt Copy Quantity: The number of copies created when holding Alt for batch copying.



Environment Settings: Configure environment settings for the current scene, including background, lighting, and weather.



Hotkey Overview: Click to view the list of available shortcut keys for the editing interface.

# IV. Details Panel

![Image 7](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhgli9r4nrni/c7e18f2a-3e68-4704-8908-1872172f1668.png)

When a unit is selected (in the Scene or in the Assets Bar), its Details Panel appears by default on the right. For Scene selections, use the Details button (top-right) to close or reopen the panel. For Assets Bar selections, after the Details panel is collapsed, reselect the unit to open it again.

# V. Mode Switching

![Image 8](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhgli9r4nrni/670487f1-dc2e-4360-ad1c-00d72f4d29ce.png)

Use the buttons at the top (see the graph) to switch Edit Modes. The four modes are:



Terrain Editing: Set up terrain, configure environment, place objects, place loot, and place creations.



Entity Placement: Place objects, loot, and creations.



Prefab Library: Create and edit Object Prefabs, Creation Prefabs, Loot Prefabs, and Prefab groups.



Combat Presets: Includes Player Templates and Class editing; Unit Status editing; Skill editing; Local Projectiles editing; and Item editing.

# VI. Camera Controls

Camera control hotkeys:

Free Camera Movement WASD
Camera Up E
Camera Down Q
Rotate Camera Right Mouse Button
Pan Camera Middle Mouse Button
Camera Zoom In/Out Mouse Scroll Wheel
Enable/Disable Camera Horizontal Lock ALT+L

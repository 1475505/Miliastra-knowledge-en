# II. Create and Save

![Image 3](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhgr26r92h8m/cbc03fff-585f-4b4b-b95e-319b970d41d7.png)

Use the New button in the Assets Bar to create content for the currently selected Edit Module.

![Image 4](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhgr26r92h8m/b32e25ba-81e4-4c8c-88f1-d0b156027b59.png)

Changes made in the Character and Class Editing module are saved automatically. Changes to Unit Statuses, Skills, Local Projectiles, and Items must be saved manually. A yellow exclamation mark in the Assets Bar (as shown in the image) indicates unsaved content. Use the Save button at the bottom (as shown) to save or save as new content.

# III. Switch Gender

![Image 5](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhgr26r92h8m/fa157ea7-acad-4496-b40c-6680a7a0ef96.png)

Use the gender toggle at the bottom to switch the model's gender in the Miliastra Sandbox.

# IV. Reset Camera

![Image 6](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhgr26r92h8m/85dcc543-f434-4de9-a9ec-eb10fa530cd3.png)

In the Combat Preset module, right-click an empty area of the scene to open the Reset Camera option. Selecting it returns the Camera to its initial state

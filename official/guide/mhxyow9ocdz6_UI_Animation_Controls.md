# UI Animation Controls

# I. UI Animation Features

_UI Animations_ can be configured in the UI Layout and display the corresponding special effect asset in the UI Layout. It is a type of UI control used to beautify the interface and enrich its visual presentation

During stage runtime, UI Animation Controls can be triggered through Node Graphs

# **II. Editing UI Animation**

![Image 2](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhxyow9ocdz6/44a2f10e-5446-4bc8-aaa0-43c3f1ee6839.png)

## **1. Add UI Animation**

In the _UI Control Group Editor window_, add the UI Control Template: UI Animation

![Image 3](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhxyow9ocdz6/7ab19f33-0e1d-4f48-8885-cb092e778e11.png)

## 2. UI Animation Settings

![Image 4](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhxyow9ocdz6/c5182a6a-d534-494f-879e-d1ab090f73d1.png)

_Select UI Animation_: Select the special effect asset to be displayed in the UI Layout

_Play Sound Effect_: If set to Yes, the sound effect included with the asset will also play

_Layer_: Only supports Above All Controls and Below All Controls. The default is Above All Controls

## 3. Preview Settings

![Image 5](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhxyow9ocdz6/6f98202b-ae12-4bab-b5e2-c3a85f45ed08.png)

Enable this in the top-right settings to keep UI animations always visible in preview while editing, regardless of layer order

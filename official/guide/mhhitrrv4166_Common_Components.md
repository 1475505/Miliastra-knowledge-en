# Common Components

_Components_ are functions that can be added to entities.

After adding components to an entity, the entity can use the corresponding functions, and you can modify the component's data and logic through the node graph.

_Common components_ can be freely added and deleted on prefabs or entities.

Some units will come with their commonly used components by default.

# I. Editing Common Components

After selecting a prefab or entity, you can view the configured components or add and delete components in the Common Components tab on the right

# II. Component Life Cycle

Components can only be added to or removed from prefabs and entities during editing, and cannot be dynamically added or removed during gameplay. Some components can be enabled or disabled through node graphs

When an entity exists in the scene, its component functions will remain active until the entity is destroyed or removed

# II. Feature Details

## 1. Create Gift Box

Click Create Gift Box to create a new Gift Box

![Image 4](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhl18n9382fu/7ad02634-6aa2-4eee-a576-478b073ed1cd.png)![Image 5](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhl18n9382fu/d1a0e838-bc97-4269-8962-c1177498293b.png)

Each Stage can contain up to 20 Gift Boxes

## 2. Base Attributes

![Image 6](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhl18n9382fu/9a8d0d7c-2b67-4348-a346-a31834f55e41.png)

_Gift Box Index_: The unique identifier of a Gift Box; must be unique. When Creators (Craftspeople) upload a save, the Gift Box IDs in the current save must be a subset of the online version's Gift Box IDs for a successful upload

_Select Icon_: Select an icon for this Gift Box from the provided icons, or upload a custom icon from your local device

_Rarity_: Indicates the quality of this Gift Box

_Unlock Condition Type_: Specifies how this Gift Box can be obtained: through Leaderboard, Ranks, or Ultimate Achievement

![Image 7](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhl18n9382fu/520c2b22-6ca5-44c6-884f-fca6d2428cee.png)



When selecting a Leaderboard, first choose which leaderboard data to use as the condition, then set the required rank to achieve

![Image 8](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhl18n9382fu/a90465f0-308f-4458-9535-000cf1ec4679.png)

![Image 9](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhl18n9382fu/40f4b1a4-9b47-43bf-9763-1f08ead75977.png)



When selecting a Rank, choose the ranking requirement to achieve

![Image 10](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhl18n9382fu/552754f2-7192-4c32-8b12-a7ffd27141c2.png)



When selecting Ultimate Achievement, no additional editing is required

![Image 11](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhl18n9382fu/90729660-8fba-4635-8472-d6af89b69b4d.png)

_Can it be obtained again after the season update?_: When enabled, players can obtain this Gift Box again when the Season updates

_Validity Period_: When disabled, the Gift Box remains permanently active. When enabled, allows setting the number of Seasons it stays active for the player

_Number of valid seasons after obtaining_: Specifies the number of Seasons this Gift Box remains active for the player after being obtained

_Description Content_: Describes this Gift Box's function

## 3. Apply Settings

Defines the Seasons during which this Gift Box can be active

![Image 12](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhl18n9382fu/8bb80231-ec92-4a88-bfde-e503764db89a.png)

_Activate?_: When disabled, this Gift Box will not be available under any circumstances. When enabled, it becomes available during the configured Season

_Display Priority_: The display order of this Gift Box in the Shop area; higher values appear first

_Permanent Effect After Activation?_: When disabled, set the effective Season via Listing Settings by choosing the corresponding Season; when enabled, specify the initial effective Season, after which the Gift Box remains listed permanently

![Image 13](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhl18n9382fu/f7683834-5b11-41f6-8b9d-73262459c797.png)

## 4. Launch Settings

Defines which Gift Boxes are listed for each Season

![Image 14](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhl18n9382fu/265bc4c7-1d6f-4752-a68a-1ed016e264e6.png)

_Select Season_: Select a Season to edit the Gift Boxes to be listed for that Season

_Add_: Add Gift Boxes to be listed in the currently selected Season

![Image 15](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhl18n9382fu/9938e2ee-9324-401c-b9e5-fd1e755ee549.png)

Click the top-left corner or uncheck to remove the Gift Box from the Season. Permanent Gift Boxes cannot be removed in Listing Settings

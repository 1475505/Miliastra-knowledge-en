# Special Number Controls

# I. Special Number Features

_Special Numbers_ can be configured in the UI Layout to support displaying various types of skill-related data

During stage runtime, Special Number controls can be triggered through Node Graphs

# **II. Editing Special Numbers**

![Image 2](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhg70rrcygcu/21e043db-ea96-44f7-8c8e-47914d19a613.png)

## **1. Add Special Numbers**

In the _UI Control Group Editor window_, add the UI Control Template - Special Number

![Image 3](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhg70rrcygcu/b27cc957-aee0-40b2-82eb-74567b514385.png)

## 2. Special Number Settings

![Image 4](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhg70rrcygcu/6a71a7a9-e8c8-4c26-8aeb-e768a35e5586.png)

Configuration Parameters Description
_Data Source_ Default is Skill
_Monitor Skill Slot_ Supports selecting any slot and monitoring skills based on the selected slot

![Image 5](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhg70rrcygcu/2b956e64-71bf-4ce7-b341-3b8df7db72e9.png)
_Data Content_ Can display Skill Remaining CD Time, Skill Usage(s), Skill Resources, and Skill Availability

![Image 6](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhg70rrcygcu/a4385345-9114-4f4c-b495-709ec3c8883d.png)
_Time Unit_ Available options are Seconds, Minutes, and Hours
_Font Size_ Sets the font size of the number shown

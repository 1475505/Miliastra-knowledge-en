# Specialized Settings

# I. What are Specialized Settings

Specialized settings provide content related to combat performance and value verification

It is a means for creators (Craftspeople) to set specialized attributes for objects/creations

# II. Editing Specialized Settings

The specialized settings for objects and creations differ, as detailed below

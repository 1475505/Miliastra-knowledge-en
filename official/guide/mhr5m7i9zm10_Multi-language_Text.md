# Multi-language Text

# I. Multilingual Text Functions

Provide creators (Craftspeople) with a way to adapt multiple languages, allowing all visible text to have corresponding multilingual translations, helping creators (Craftspeople) expand their stage's user base

# II. Multilingual Text Editing

## 1. Entry

Click the system menu button in the top left corner to open the menu interface

Click on [Manage Multi-language Text] to enter the multi-language text management interface, where you can configure different language packs for the game to support more languages

## 2. Detailed Introduction

![Image 4](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhr5m7i9zm10/7b8d471e-ac63-44bc-89bc-6e37556aa519.png)

_Search_: Used to search for corresponding text, the search content is the text in the original language

_Source_: Used to display the source of the text, showing where it is used, can be filtered

![Image 5](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhr5m7i9zm10/2012f4ee-23d8-43bd-a51f-c8bd9e18bf23.png)_Need Translation?_: Used to indicate whether this text needs to be translated

Yes: The text must be translated into every supported language. Upload will fail if there is any untranslated content.

No: This text is marked as not requiring translation

The following column displays the original language + supported languages for this save file. Click [Supported Languages] in the bottom right to select the multiple languages supported by this save file

![Image 6](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhr5m7i9zm10/dc8a453f-7bbd-4894-af85-19e403becdfe.png)![Image 7](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhr5m7i9zm10/9fc93c18-f4d3-4236-bc75-aa757ce04912.png)

_Refresh List_: Retrieve text content from the archive again _Export/Import_: Export this list as a CSV text file for external editing, then import it back into this list

Import validation rules:

Headers must be exactly the same

The number of rows is not validated, only identical source language text will be filled in

The "Translate" column is not validated, content in the table will be filled in automatically

# Skill Animation

# I. Definition of Skill Animation

Skill animations are animation assets created for combat actions performed by entities

# II. Character Skill Animation

## 1. Features of Character Skill Animations

Character skill animations require a character entity as their base, and each skill animation playback must have a specific character entity as its carrier.

Character skill animations depend on the activation of custom skills.

Currently, character skill animations are divided into three categories based on the characteristics of skill usage: _Normal Attack,_ _Release Continuously_ _, and Movement._

## 2. Reference to Character Skill Animations

When editing skills, Craftspeople can define the animation effects of the character after the skill is released by referencing the character skill animation.

For specific reference methods, please refer to [Skills](https://act.hoyoverse.com/ys/ugc/tutorial//detail/mho81frl33im)

# III. Creation Skill Animations

## 1. Characteristics of Creation Skill Animations

Creation skill animations must rely on a creation entity; each animation requires a specific creation entity as its medium.

The playback of creation skill animations depends on the release of custom skills.

Currently, creation skill animations are divided into two categories based on their usage characteristics: _normal and continuous._

## 2. Referencing Creation Skill Animations

During skill editing, Craftspeople can define the animation effects of the creation after the skill is released by referencing creation skill animations.

For specific reference methods, please refer to [Skills](https://act.hoyoverse.com/ys/ugc/tutorial//detail/mho81frl33im)

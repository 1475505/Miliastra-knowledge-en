# Miliastra Sandbox

# I. Miliastra Sandbox Entry Point



Automatically opens when entering Miliastra Sandbox



After closing, open Miliastra Sandbox from the bottom-left of the Start Menu

![Image 2](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhgwm3rdh6tq/eb11e44b-2150-4b2a-8b90-5c14bd6900db.png)

# II. Miliastra Sandbox UI Appearance



The left standalone window is the Resource Explorer



The right standalone window is the Node Graph Explorer



When multiple tabs are present, you can drag to reorder them or pull one out to create a sub-window

![Image 3](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhgwm3rdh6tq/ad751038-2a8e-4241-9d17-6e4cebf20482.png)

# III. Resource Explorer



Displays objects, prefabs, and other resources already present in the current session

![Image 4](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhgwm3rdh6tq/84a5d0cd-6d84-49c6-b1c9-e3ffa3750c73.png)

# IV. Server Node Graph Explorer



The Server Node Graph Explorer opens by default when Miliastra Sandbox is enabled



The upper section lists all currently created Server Node Graphs



The lower section lists resource objects that can mount Server Node Graphs



Main functions of the Server Node Graph Explorer:



Create and manage Entity Node Graphs, Status Node Graphs, Class Node Graphs, and Item Node Graphs



You can import Server Node Graphs from external Node Graph assets

![Image 5](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhgwm3rdh6tq/11be1836-d17c-4409-b0e7-87032bb8c259.png)



Open the following via [Window] in the top Menu Bar:



Resource Explorer



Client Node Graph Resource Explorer



Node Explorer



Log



Server Signal Explorer



Load Detection

![Image 6](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhgwm3rdh6tq/5db45375-204a-461e-a73c-547c71e0d73a.png)

# V. Client Node Graph Resource Explorer



Main functions of the Client Node Graph Resource Explorer:



Create and manage character skill node graphs, creation skill node graphs, creation status node graphs, creation status decision node graphs, and local filter node graphs.



Local Filters include two types: Boolean filters and Integer filters



You can import Client Node Graphs from external Node Graph assets

![Image 7](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhgwm3rdh6tq/d622593c-c3e9-4b18-9a48-408260b94b0f.png)

# VI. Nod e Explorer



Main functions of the Node Explorer:



View all current node categories and node details



Create and manage Composite Nodes



Import Composite Nodes from external node assets

![Image 8](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhgwm3rdh6tq/cd4350b2-8a67-44bb-994a-7c584cf5045a.png)

# VII. Log



Main logging functions:



Select specific node graphs to record runtime information in real time while the stage is running

![Image 9](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhgwm3rdh6tq/e446a602-e4df-4c39-8247-9bd2dc67958e.png)

Main Functions of Server Signals:



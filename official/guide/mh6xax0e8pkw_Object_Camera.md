# Object Camera

# I. Functions of the Object Camera Component

This component lets you mount additional cameras to dynamic objects/creations and switch between cameras while the stage is running for better visual presentation

# II. Editing Object Camera Components

## **1. Add Components**

![Image 2](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mh6xax0e8pkw/0b977207-e664-4512-8e30-44dd3a7ab93f.png)

(1) In the Entity/Prefab Editing interface, open the Component Editing Tab

(2) Click "Add Common Component" below, then select and click "Object Camera" to add it

(3) Click "Edit Details" to expand the editing tab

## **2. Editing the Object Camera Component**

![Image 3](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mh6xax0e8pkw/cb914865-b6ee-49d5-a2a1-81ac6e9ba143.png)

Click + New Object Camera, then use Select Camera to choose a preconfigured Object Camera Template

Use [Manage Cameras] to open [Main Camera Management], where you can edit the cameras

Select a camera, then use [Enter Preview Mode] to preview it

![Image 4](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mh6xax0e8pkw/2200c739-72c9-47f0-a50d-b1b154020c8d.png)

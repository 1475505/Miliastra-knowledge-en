# Prefab Library

# I. Prefab Library Entry Point

The third icon in the top Tab bar

![Image 2](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhppo7iyysuc/579478e0-ea26-4b3d-83c5-ba6a44de1c6b.png)

# II. Prefab Library Interface Style

![Image 3](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhppo7iyysuc/9fd7f05d-4a74-4a48-845a-05e192bfe96f.png)



The Prefab Library's Assets Bar is on the left

# III. Assets Bar Details

## 1. Custom Prefabs



Includes an Uncategorized tab by default



Click Add Tab to create a Custom tab

## 2. Basic Module Prefabs (Prefabricated Components)



Includes Dynamic Prefabs and Static Prefabs



Dynamic Prefabs include: process objects, gameplay mechanisms, and others



Static Prefabs include: trees, vegetation, terrain features, etc.

## 3. Asset Library



Show importable prefabs based on local assets

# IV. How to Use the Prefab Library

## 1. Create Prefabs

### (1) Asset Panel — Basic Module tab: create new Prefabs



Click the Asset icon to create a prefab at the center of the camera view



Drag the Asset icon to create a prefab at the cursor position

![Image 4](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhppo7iyysuc/ad096d4e-f3d7-41dd-b7df-957265d562c3.png)

### (2) Copy-Paste Custom Prefabs via right-click to create new prefabs



Copy an existing Custom Prefab from the right-click menu



Right-click a blank area and choose Paste

![Image 5](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhppo7iyysuc/f7268d15-3320-41ad-ac33-23e4c3fc318f.png)

![Image 6](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhppo7iyysuc/364b9fc9-0b96-4986-9a3b-2df333f98abd.png)

### (3) Appearance of newly created, unsaved prefabs



A green plus icon appears on the object as a UI indicator

![Image 7](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhppo7iyysuc/1f17db0d-8c77-49a8-8905-60b50586c664.png)

## 2. Edit Prefab



Click the prefab you want to edit to open its Details page on the right. On the Details page, you can edit the Prefab's Attributes, Components, Node Graphs, and Combat Content, using the same methods as object editing



Asset Panel Prefab Right-Click Menu

![Image 8](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhppo7iyysuc/a3f49217-1dd4-45d6-b120-aa1f204da2e9.png)



Prefab Icon Menu on the Details Page

![Image 9](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhppo7iyysuc/bb2d8afc-5ad5-44cc-bae9-6d969300c20d.png)



Appearance of Modified, Unsaved Prefabs



When a prefab has new additions or edits, the corresponding module title shows yellow highlighted text, and the highlight disappears after saving

![Image 10](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhppo7iyysuc/929ea3db-7d93-44e8-bfa8-274574870fb8.png)



When a Custom Prefab has additions or edits, the object shows a yellow exclamation-mark UI indicator, and the corresponding icon in the left Assets Bar is also marked with a yellow exclamation mark

![Image 11](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhppo7iyysuc/09f2b59f-d560-4545-94e1-9af8f37269f7.png)



Prefabricated Components will still display a green plus icon when modified

![Image 12](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhppo7iyysuc/808b0fe4-e2fd-4de0-be38-bba9e5a2a188.png)

## 3. Save Prefab



Create a new unsaved prefab



Click [Save] or [Save As Prefab] in the bottom menu or the right-click menu to save the prefab



You can rename the prefab and choose a tab when saving

![Image 13](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhppo7iyysuc/e1cea7c7-87fa-4216-b186-7e1506ab03c8.png)

![Image 14](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhppo7iyysuc/45abe26f-37c9-42b1-83da-ca8abead2b1e.png)



Modified Custom Prefabs



Click [Save As Prefab] in the bottom menu or right-click menu to save the prefab



Click [Save As Prefab] to create a new prefab



Click [Overwrite and Save Prefab] to save over the original prefab



Note: When saving with overwrite, all objects that reference this prefab in the Entity Placement tab will be overwritten. Any additional modifications made to those objects in the Entity Placement tab will not be overwritten

![Image 15](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhppo7iyysuc/e09f794e-9a34-4963-b7a3-104054eb755c.png)

![Image 16](https://act-webstatic.hoyoverse.com/ugc-tutorial/knowledge/sea/en-us/mhppo7iyysuc/ee2a416a-47dd-4b84-a1b0-8e7df463a494.png)

## 4. Using Prefabs



In-game Use






